== Description ==

Provides a UI for manipulating the custom hooks in the Oenology Theme. Requires Oenology Theme version 2.0, and compatible up to version 3.0.

== Installation ==

Manual installation:

1. Upload the `oenology-hooks` folder to the `/wp-content/plugins/` directory

Installation using "Add New Plugin"

1. From your Admin UI (Dashboard), use the menu to select Plugins -> Add New
2. Search for 'Oenology Hooks'
3. Click the 'Install' button to open the plugin's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the plugin through the 'Plugins' menu in WordPress
2. From your Admin UI (Dashboard), use the menu to select Appearance -> Hooks 
3. Configure settings, and save
